    Updated: 2009-03-23 00:20


    д»ҠеӣһгҒҜгҖҒжҳЁж—Ҙ пјҰпҪ…пҪ„пҪҸпҪ’пҪҒпј— гӮ’гӮӨгғігӮ№гғҲгғјгғ«гҒ—гҒҹ KURO-BOX/HG гӮ’ пјҰпҪ…пҪ„пҪҸпҪ’пҪҒпјҳ гҒ«гӮўгғғгғ—гӮ°гғ¬гғјгғүгҒ—гҒҰгҒҝгҒҫгҒ—гҒҹгҖӮ

    гҒ„гҒӨгӮӮгҒӘгҒҢгӮүгҖҒгҒ“гӮҢгҒҜгҒӮгҒҸгҒҫгҒ§гӮӮиҮӘеҲҶз”ЁгҒ®иЁҳйҢІгҒ§гҒҷгҖӮиӢҘе№Із’°еўғзӯүз•°гҒӘгӮӢе ҙеҗҲгӮӮгҒӮгӮӢгҒӢгӮӮгҒ—гӮҢгҒҫгҒӣгӮ“гҖӮгҒ”дәҶжүҝдёӢгҒ•гҒ„гҖӮ

    дҪңжҘӯиЁҳйҢІ

    гҖҗеүҚжҸҗжқЎд»¶гҖ‘ гғ»пјҰпҪ…пҪ„пҪҸпҪ’пҪҒеҢ–гҒҷгӮӢзҺ„з®ұгҒҜгҖҢ KURO-BOX/HG гҖҚгҖӮ гғ»гҖҢ KURO-BOX/HG гҖҚгҒ«гҒҜпј“пј’пјҗпј§пјўгҒ®гғҸгғјгғүгғҮгӮЈгӮ№гӮҜгӮ’зө„гҒҝиҫјгӮ“гҒ§гҒ„гӮӢгҖӮ гғ»гҖҢ KURO-BOX/HG гҖҚгҒ®пј©пј°гӮўгғүгғ¬гӮ№гҒҜ [ 192.168.11.2 ] гғ»гҖҢ KURO-BOX/HG гҖҚгҒ«гҖҢ гҒ§гҒҚгӮӢпјҒзҺ„з®ұ FedoraеҢ–пјҒпјҒ гҖҚгҒ®гӮӨгғігӮ№гғҲгғјгғ«гӮӯгғғгғҲгҖҢ F4K Install Kit гҖҚгҒ§пјҰпҪ…пҪ„пҪҸпҪ’пҪҒпј—е°Һе…ҘжёҲгҒҝпјҶгҖҢ yum гҖҚгҒҢжңҖж–°зүҲгҒ«е…ҘгӮҢжӣҝгҒҲжёҲгҒҝгҖӮ гғ»гӮӨгғігӮ№гғҲгғјгғ«дҪңжҘӯгӮ’е®ҹиЎҢгҒ•гҒӣгӮӢгғһгӮ·гғігҒҜпј·пҪүпҪҺпҪ„пҪҸпҪ—пҪ“пјёпј°пјҲпј°пҪ…пҪҺпҪ”пҪүпҪ•пҪҚпјӯпјү

    пј‘пјҺзҺ„з®ұгҒёгҒ®жҺҘз¶ҡ

    гғӯгғјгӮ«гғ«гғһгӮ·гғігҒӢгӮү KURO-BOX/HG гҒёгғӘгғўгғјгғҲжҺҘз¶ҡгҒ—гҖҒrootгғҰгғјгӮ¶гҒ«гҒӘгӮӢгҖӮ пјҲеҪ“ж–№гҒҜгҖҒгҖҢ Poderosa гҖҚгҒЁгҒ„гҒҶгҒ®гӮ’дҪҝз”ЁгҒ—гҒҰгҒ„гҒҫгҒҷпјү

    пј’пјҺгғ‘гғғгӮұгғјгӮёгғҖгӮҰгғігғӯгғјгғү

    пј‘пј•пјҺгғӘгғ–гғјгғҲ

    пј‘пј–пјҺгғҗгғјгӮёгғ§гғізўәиӘҚ

    еҶҚеәҰгғӘгғўгғјгғҲжҺҘз¶ҡгҒ—rootгғҰгғјгӮ¶гҒ«гҒӘгӮҠгҖҒд»ҘдёӢгҒ®гӮҲгҒҶгҒ«е…ҘеҠӣгҒ—гҒҰгҒҝгӮӢгҖӮ
