Cygwin で top コマンドや free コマンドなどが使用できない場合があります。

インストールされていないからであり、procps というパッケージをインストールすることで解決できます。

<!--more-->

### 導入メモ

#### ０．procps について

Cygwin に procps というパッケージを導入することで、以下のコマンド使用できるようになります。

``` text
prockill  procps    sysctl    free      pgrep     pkill
pmap      pwdx      skill     slabtop   snice     tload
top       uptime    vmstat    w         watch
```

以下を参照。

- [http://cygwin.com/packages/procps/](http://cygwin.com/packages/procps/ "http://cygwin.com/packages/procps/")

#### １．導入済み確認

以下のコマンドで Cygwin に既に procps が導入済みかどうかを確認します。

``` bash 
$ cygcheck -c | grep procps
```

なにも表示されなければ、未インストールです。
apt-cyg を導入しているなら、以下のコマンドでも確認できます。

``` bash 
$ apt-cyg find procps
```

#### ２．procps インストール

Cygwin の setup.exe を起動して、[System] の procps をインストールするだけです。
apt-cyg を導入しているなら、Cygwin 上から以下のコマンドでもインストールできます。

``` bash 
$ apt-cyg install procps
```

#### ３．導入完了確認

以下のコマンドで Cygwin に procps がインストールできたかを確認します。

``` bash 
$ cygcheck -c | grep procps
procps               3.2.7-1              OK
```

このように表示されれば、インストールできています。
apt-cyg を導入しているなら、以下のコマンドでも確認できます。

``` bash 
$ apt-cyg find procps
```
