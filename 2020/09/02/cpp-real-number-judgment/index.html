    C++ - 実数判定（正規表現）！

    Updated: 2020-09-02 00:20


    C++ で文字列が実数か否かを正規表現を使用して判定する方法についてです。
    （別途作成予定のツールで部品として使用するための事前準備）

    0. 前提条件

    • Debian GNU/Linux 10.3 (64bit) での作業を想定。
    • GCC 9.2.0 (G++ 9.2.0) (C++17) でのコンパイルを想定。

    1. ソースコードの作成

    • 正規表現を使用して判定する。
    • stof 等を利用した判定方法もあるが、期待しない例外が発生したり、場合分けが面倒になる。

    File: is_real.cpp

    • Gist - C++ source code to judge if a string is an real number.

    2. ソースコードのコンパイル

    $ g++ -std=c++17 -Wall -O2 --pedantic-errors -o is_real is_real.cpp

    3. 動作確認

    $ ./is_real
n? 1.2
1.2: REAL NUMBER
---
n? 1.
1.: REAL NUMBER
---
n? .1
.1: REAL NUMBER
---
n? -2.3
-2.3: REAL NUMBER
---
n? -.3
-.3: REAL NUMBER
---
n? -4.
-4.: REAL NUMBER
---
n? 1.a
1.a: NOT REAL NUMBER
---
n? -12345.456789
-12345.456789: REAL NUMBER
---
n? -4.56O143
-4.56O143: NOT REAL NUMBER
---
n?
    • 最後の小数点以下第3位の O はアルファベット大文字の「オー」

    以上。
