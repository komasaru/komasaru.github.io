    пј·пҪҸпҪ’пҪ„пј°пҪ’пҪ…пҪ“пҪ“ пјҚ пј·пј° пјіпҪ•пҪҗпҪ…пҪ’ пјЈпҪҒпҪғпҪҲпҪ… е°Һе…ҘпјҒ

    Updated: 2009-12-16 00:20


    жңҖиҝ‘гҖҒгғ–гғӯгӮ°гғҡгғјгӮёпјҲпј·пҪҸпҪ’пҪ„пј°пҪ’пҪ…пҪ“пҪ“пјүгҒ®иЎЁзӨәгҒҢйҮҚгҒ„гӮҲгҒҶгҒ«ж„ҹгҒҳгӮӢгӮҲгҒҶгҒ«гҒӘгҒЈгҒҹгҒ®гҒ§гҖҒ иЎЁзӨәгӮ’и»ҪгҒҸгҒҷгӮӢгғ—гғ©гӮ°гӮӨгғігӮ’е°Һе…ҘгҒ—гҒҰгҒҝгҒҫгҒ—гҒҹгҖӮ

    гҒ„гӮҚгҒ„гӮҚгҖҒиӘҝгҒ№гҒҰгҒҝгӮӢгҒЁгҖҒ гҖҢ пј·пј° пјіпҪ•пҪҗпҪ…пҪ’ пјЈпҪҒпҪғпҪҲпҪ… гҖҚгҒЁгҒ„гҒҶгҒ®гҒҢиүҜгҒ•гҒқгҒҶпјҒ дёҖж—ҰиЎЁзӨәгҒ—гҒҹгғҡгғјгӮёгӮ’гӮӯгғЈгғғгӮ·гғҘгҒ—гҒҰгҖҒж¬ЎеӣһгҒӢгӮүиЎЁзӨәгҒҢгӮ№гғ”гғјгғүгӮўгғғгғ—гҒҷгӮӢгҒЁгҒ„гҒҶгӮӮгҒ®гҖӮ

    гҒӮгӮҢгҒ“гӮҢгҖҒгӮөгӮӨгғҲгҒ§иӘҝгҒ№гҒӘгҒҢгӮүе°Һе…ҘгҒ—гҒҰгҒҝгҒҫгҒ—гҒҹгҖӮ

    е°Һе…ҘиЁҳйҢІ

    пј‘пјҺгҖҢ wp-config.php гҖҚз·ЁйӣҶ

    гӮӯгғЈгғғгӮ·гғҘгӮ’дҪҝз”ЁгҒ§гҒҚгӮӢгӮҲгҒҶгҒ«гҒҷгӮӢгҒҹгӮҒгҒ«гҖҒгҖҢ wp-config.php гҖҚгғ•гӮЎгӮӨгғ«гҒ«д»ҘдёӢгҒ®дёҖиЎҢгӮ’иҝҪеҠ гҒҷгӮӢгҖӮ

    define('WP_CACHE', true);гҖҖвҶҗгҖҖиҝҪеҠ 

if ( !defined('ABSPATH') )гҖҖгҒ“гҒ®иЎҢгӮҲгӮҠдёҠгҒ«иЁҳиҝ°гҒҷгӮӢгҒ“гҒЁ

    вҖ»жүҖжңүиҖ…гҖҒгғ‘гғјгғҹгғғгӮ·гғ§гғігҒ«жіЁж„ҸпјҒ

    пј’пјҺгҖҢ WP Super Cache гҖҚгғҖгӮҰгғігғӯгғјгғү

    гҒ“гҒЎгӮү[ http://wordpress.org/extend/plugins/wp-super-cache/ ]гҒӢгӮү гҖҢ wp-super-cache.0.9.8.zip гҖҚгӮ’гғҖгӮҰгғігғӯгғјгғүпјҶеұ•й–Ӣ вҖ»еҪ“иЁҳдәӢеҹ·зӯҶжҷӮзӮ№гҒ§гҒҜгҖҒпј¶пҪ…пҪ’пјҺпјҗпјҺпјҷпјҺпјҳ гҒҢжңҖж–°гҖӮ

    пј“пјҺгҖҢ WP Super Cache гҖҚгӮўгғғгғ—гғӯгғјгғү

    дёҠиЁҳгҒ§еұ•й–ӢгҒ—гҒҰгҒ§гҒҚгҒҹгғ•гӮ©гғ«гғҖгҖҢ wp-super-cache гҖҚгӮ’гӮөгғјгғҗгҒ®гҖҢ plugins гҖҚгғ•гӮ©гғ«гғҖгҒёгӮўгғғгғ—гғӯгғјгғүгҒҷгӮӢгҖӮ вҖ»жүҖжңүиҖ…гҖҒгғ‘гғјгғҹгғғгӮ·гғ§гғігҒ«жіЁж„ҸпјҒ

    пј”пјҺгғ‘гғјгғҹгғғгӮ·гғ§гғіеӨүжӣҙ

    д»ҠгҒҫгҒ§гҒ®дҪңжҘӯгҒ§гғ‘гғјгғҹгғғгӮ·гғ§гғігӮ’гҒ„гҒҳгҒЈгҒҰгҒ„гӮӢгҒӢгӮӮгҒ—гӮҢгҒҫгҒӣгӮ“гҒҢгҖҒ гҒ“гҒ“гҒ§гҖҢ wordpress гҖҚгғ»гҖҢ wordpress/wp-content гҖҚгғ•гӮ©гғ«гғҖгҒ®гғ‘гғјгғҹгғғгӮ·гғ§гғігӮ’гҖҢ пј—пј—пј— гҖҚгҒ«еӨүжӣҙгҒ—гҒҰгҒҠгҒҸгҖӮ

    пј•пјҺгғ—гғ©гӮ°гӮӨгғіжңүеҠ№еҢ–

    пј·пҪҸпҪ’пҪ„пј°пҪ’пҪ…пҪ“пҪ“гҒ®гғ—гғ©гӮ°гӮӨгғіз®ЎзҗҶз”»йқўгҒ§гҖҒгҖҢ WP Super Cache гҖҚгӮ’дҪҝз”ЁгҒҷгӮӢгӮҲгҒҶгҒ«гҒҷгӮӢгҖӮ гҒҷгӮӢгҒЁгҖҒ гҖҢ WP Super Cache is disabled. Please go to the plugin admin page to enable caching. гҖҚ гҒЁгғЎгғғгӮ»гғјгӮёгҒҢиЎЁзӨәгҒ•гӮҢгӮӢгҒ®гҒ§гҖҒ гҖҢ plugin admin page гҖҚгҒ®гғӘгғігӮҜгӮ’гӮҜгғӘгғғгӮҜгҒҷгӮӢгҖӮ гҖҢ ON WP Cache and Super Cache enabled гҖҚгҒ«гғҒгӮ§гғғгӮҜгҖӮ гҖҢ Clear all cache files when a post or page is published. (This may significantly slow down saving of posts.) гҖҚгҒ«гғҒгӮ§гғғгӮҜгҖӮ гҖҢ Update Status гҖҚгҒ®гғңгӮҝгғігҒ§иЁӯе®ҡгӮ’дҝқеӯҳгҒҷгӮӢгҖӮ гҒҷгӮӢгҒЁгҖҒгҖҢ wordpress/wp-content гҖҚгғ•гӮ©гғ«гғҖгҒ«

    • гҖҢ cache гҖҚгғ•гӮ©гғ«гғҖ

    • гҖҢ wp-cache-config.php гҖҚгғ•гӮЎгӮӨгғ«

    • гҖҢ advanced-cache.php гҖҚгғ•гӮЎгӮӨгғ«

    гҒҢдҪңжҲҗгҒ•гӮҢгӮӢгҖӮ

    пј–пјҺгғ‘гғјгғҹгғғгӮ·гғ§гғіеӨүжӣҙ

    гғ‘гғјгғҹгғғгӮ·гғ§гғігӮ’гҖҢ пј—пј—пј— гҖҚгҒ«гҒ—гҒҰгҒ„гҒҹгҖҢ wordpress гҖҚгғ»гҖҢ wordpress/wp-content гҖҚгғ•гӮ©гғ«гғҖгӮ’гҖҢ пј—пј•пј• гҖҚгҒ«жҲ»гҒҷгҖӮ

    пј—пјҺгӮӯгғЈгғғгӮ·гғҘзўәиӘҚ

    гғ–гғӯгӮ°гғҡгғјгӮёгӮ’иЎЁзӨәгҒ•гҒӣгҒҰгҒҝгҒҰгҖҒ з®ЎзҗҶз”»йқўгҒ®[ иЁӯе®ҡ ] - [ WP Super Cache ] - [ Cache Contents ] гҒ«гҖҒ д»ҘдёӢгҒ®гӮҲгҒҶгҒ«иЎЁзӨәгҒ•гӮҢгҒҰгӮҢгҒ°гӮӯгғЈгғғгӮ·гғҘжҲҗеҠҹпјҒ

    гҒ—гҒ°гӮүгҒҸгҖҒдҪҝз”ЁгҒ—гҒҰж§ҳеӯҗгӮ’иҰӢгҒҰгҒҝгӮҲгҒҶгҒЁжҖқгҒ„гҒҫгҒҷгҖӮ
