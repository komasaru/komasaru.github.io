    MySQL пјҚ гҖҢCanвҖҷt connect to MySQL server on вҖҳlocalhostвҖҷ (10048)гҖҚгӮЁгғ©гғјпјҒ

    Updated: 2011-02-19 00:20


    жңҖиҝ‘гҖҒпј·пҪүпҪҺпҪ„пҪҸпҪ—пҪ“пјёпј°гғһгӮ·гғігҒ§гҖҢпјІпҪ•пҪӮпҪҷ пҪҸпҪҺ пјІпҪҒпҪүпҪҢпҪ“гҖҚпјӢгҖҢпјӯпҪҷпјіпјұпј¬гҖҚгҒ®гӮўгғ—гғӘгӮ’дҪңжҲҗгҒ—гҒҰгҒ„гӮӢгҒЁд»ҘдёӢгҒ®гӮҲгҒҶгҒӘгӮЁгғ©гғјгҒҢзҷәз”ҹгҒҷгӮӢгӮҲгҒҶгҒ«гҒӘгӮҠгҒҫгҒ—гҒҹгҖӮ

    CanвҖҷt connect to MySQL server on 'localhost' (10048)

    иӘҝгҒ№гҒҰгҒҝгӮӢгҒЁгҖҒгҒ©гҒҶгӮ„гӮүпј·пҪүпҪҺпҪ„пҪҸпҪ—пҪ“гҒ гҒ‘гҒ®е•ҸйЎҢгӮүгҒ—гҒ„гҒ§гҒҷгҖӮпјҲпј¬пҪүпҪҺпҪ•пҪҳзӯүгҒ§гҒҜе•ҸйЎҢгҒӘгҒ„гӮҲгҒҶгҒ§гҒҷпјү

    д»ҠеҫҢгҒ®гҒҹгӮҒгҒ«иЁҳйҢІгҒЁгҒ—гҒҰж®ӢгҒ—гҒҰгҒҠгҒҚгҒҫгҒҷгҖӮ

    еҺҹеӣ 

    пј·пҪүпҪҺпҪ„пҪҸпҪ—пҪ“пјёпј°гӮ„пј·пҪүпҪҺпҪ„пҪҸпҪ—пҪ“пј’пјҗпјҗпј“пјіпҪ…пҪ’пҪ–пҪ…пҪ’гҒӘгҒ©гҒ®пј·пҪүпҪҺпҪ„пҪҸпҪ—пҪ“пјҜпјігҒ§гҒҜгҖҒдёҖжҷӮзҡ„гҒӘжҺҘз¶ҡгҒ®гҒҹгӮҒгҒ«дҪҝз”ЁеҸҜиғҪгҒӘйҖҡдҝЎгғқгғјгғҲгҒЁгҒ—гҒҰгҖҒ1025з•ӘгҒӢгӮү5000з•ӘгҒҫгҒ§гӮ’дҪҝз”ЁгҒҷгӮӢиЁӯе®ҡгҒ«гҒӘгҒЈгҒҰгҒ„гӮӢгҖӮ гҒ“гҒ®гҒҹгӮҒгҖҒгӮ»гғғгӮ·гғ§гғіж•°гҒҢеў—еӨ§гҒҷгӮӢгҒЁдҪҝз”ЁеҸҜиғҪгҒӘгғқгғјгғҲгҒҢдёҚи¶ігҒ—гҖҒпјҜпјіеҒҙгҒ§ж–°гҒ—гҒ„жҺҘз¶ҡгӮ’еҸ—гҒ‘д»ҳгҒ‘гӮүгӮҢгҒӘгҒҸгҒӘгӮӢгҒ“гҒЁгҒҢгҒӮгӮӢгҖӮ вҖ»е…·дҪ“зҡ„гҒ«гҒҜгҖҒпј·пҪүпҪҺпҪ„пҪҸпҪ—пҪ“пјҜпјігҒӢгӮүпјӯпҪҷпјіпјұпј¬гӮөгғјгғҗгҒ«жҜҺз§’пј’пјҗеӣһзЁӢеәҰд»ҘдёҠжҺҘз¶ҡгғ»еҲҮж–ӯгӮ’з№°гӮҠиҝ”гҒҷгҒЁж•°еҚғеӣһгҒҸгӮүгҒ„гҒ§жҺҘз¶ҡгӮЁгғ©гғјгҒ«гҒӘгӮӢгҖӮгҒЁгҒ„гҒҶгҒ“гҒЁгҖӮ

    и§Јжұәзӯ–

    з§ҒгҒҢиӘҝгҒ№гҒҹзөҗжһңгҖҒд»ҘдёӢгҒ®пј’йҖҡгӮҠгҒ®и§Јжұәж–№жі•гҒҢгҒӮгӮӢгӮҲгҒҶгҒ§гҒҷгҖӮ

    и§Јжұәж–№жі•пј‘

    пјӯпҪҷпјіпјұпј¬гӮөгғјгғҗгҒёгҒ®жҺҘз¶ҡеӣһж•°гҒҢжҜҺз§’пј’пјҗеӣһзЁӢеәҰд»ҘдёӢгҒ«гҒӘгӮӢгӮҲгҒҶгҒ«гғ—гғӯгӮ°гғ©гғ гӮ’жӣёгҒҚжҸӣгҒҲгӮӢгҖӮ вҖ»е…·дҪ“зҡ„гҒ«гҒҜгҖҒгғ«гғјгғ—гҒ—гҒӘгҒҢгӮүдҪ•еҚғеӣһгҒЁпјӯпҪҷпјіпјұпј¬гӮөгғјгғҗгҒёгӮўгӮҜгӮ»гӮ№гҒ•гҒӣгӮӢгҒ®гҒ§гҒҜгҒӘгҒҸгҖҒгҒӘгӮӢгҒ№гҒҸгҒҫгҒЁгӮҒгҒҰгӮўгӮҜгӮ»гӮ№гҒҷгӮӢгӮҲгҒҶгҒ«гҒҷгӮӢгҖӮ

    и§Јжұәж–№жі•пј’

    пј·пҪүпҪҺпҪ„пҪҸпҪ—пҪ“гҒ®гғ¬гӮёгӮ№гғҲгғӘиЁӯе®ҡгӮ’еӨүжӣҙгҒҷгӮӢгҖӮ вҖ»гғ¬гӮёгӮ№гғҲгғӘж“ҚдҪңгҒҜеҚұйҷәгҒҢдјҙгҒ„гҒҫгҒҷгҖӮеҝ…гҒҡдҪңжҘӯеүҚгҒ«гғҗгғғгӮҜгӮўгғғгғ—гӮ’еҸ–гӮҠгҖҒеҗ„иҮӘгҒ®иІ¬д»»гҒ§дҪңжҘӯгҒҷгӮӢгҒ“гҒЁгҖӮ гғ¬гӮёгӮ№гғҲгғӘгӮ’з·ЁйӣҶгҒҷгӮӢгҒ“гҒЁгҒ«гӮҲгӮӢеҰӮдҪ•гҒӘгӮӢе•ҸйЎҢгҒ«еҜҫгҒ—гҒҰгӮӮеҪ“ж–№гҒҜдҝқиЁјгӮ’гҒ„гҒҹгҒ—гҒӢгҒӯгҒҫгҒҷгҖӮ

    вҖ»гҒҫгҒҹгҖҒеҪ“ж–№гҒҜеүҚиҝ°гҒ®гҖҢи§Јжұәж–№жі•пј‘гҖҚгҒ§еҜҫеҝңгҒ§гҒҚгҒҹгҒҹгӮҒгҖҒе®ҹйҡӣгҒ«гҒҜгҒ“гҒ®гҖҢи§Јжұәж–№жі•пј’гҖҚгҒҜи©ҰгҒ—гҒҰгҒ„гҒҫгҒӣгӮ“гҖӮ

    гғ¬гӮёгӮ№гғҲгғӘгӮЁгғҮгӮЈгӮҝгҒ§ж¬ЎгҒ®пј’гғөжүҖгҒ®гғ¬гӮёгӮ№гғҲгғӘгӮ’еӨүжӣҙгҒ—гҒҫгҒҷгҖӮпјҲеҖӨгҒҢеӯҳеңЁгҒ—гҒӘгҒ„е ҙеҗҲгҒ«гҒҜиҝҪеҠ гҒҷгӮӢпјү

    пј‘пјҺгҖҢ MaxUserPort гҖҚ гҖҢHKEY_LOCAL_MACHINEгҖҚ-гҖҢSYSTEMгҖҚ-гҖҢCurrentControlSetгҖҚ-гҖҢServicesгҖҚ-гҖҢTcpipгҖҚ-гҖҢParametersгҖҚ гҒЁгҒҹгҒ©гҒЈгҒҰгҒ„гҒҚгҖҒгҖҢ MaxUserPort гҖҚгҒ®еҖӨгӮ’еӨүжӣҙгҒҷгӮӢгҖӮпјҲпјӯпјЎпјёгҒҜгҖҢ65534гҖҚпјү

    вҖ»гҖҢ MaxUserPort гҖҚгҒҜгҖҒдҪҝз”ЁеҸҜиғҪгҒӘжңҖеӨ§гғқгғјгғҲз•ӘеҸ·гҖӮпјҲгғҮгғ•гӮ©гғ«гғҲгҒҜгҖҢ5000гҖҚпјү

    пј’пјҺгҖҢ TcpTimedWaitDelay гҖҚ гҖҢHKEY_LOCAL_MACHINEгҖҚ-гҖҢSYSTEMгҖҚ-гҖҢCurrentControlSetгҖҚ-гҖҢServicesгҖҚ-гҖҢTcpipгҖҚ-гҖҢParametersгҖҚ гҒЁгҒҹгҒ©гҒЈгҒҰгҒ„гҒҚгҖҒгҖҢ TcpTimedWaitDelay гҖҚгҒ®еҖӨгӮ’еӨүжӣҙгҒҷгӮӢгҖӮ

    вҖ»гҖҢ TcpTimedWaitDelay гҖҚгҒҜгҖҒгҖҢTIME_WAITгҖҚгӮ№гғҶгғјгӮҝгӮ№гҒ®з¶ҷз¶ҡжҷӮй–“гҖӮпјҲгғҮгғ•гӮ©гғ«гғҲгҒҜгҖҢ240пјҲз§’пјүгҖҚпјү жңҖйҒ©гҒӘеҖӨгҒҜз’°еўғгҒ«гӮҲгӮҠз•°гҒӘгӮӢгҒҹгӮҒгҖҒеҫҗгҖ…гҒ«ж•°еҖӨгӮ’жёӣгӮүгҒ—йҒ©еҲҮгҒӘеҖӨгҒ«иӘҝж•ҙгҒҷгӮӢгҖӮпјҲжҺЁеҘЁгҒҜгҖҢ60пјҲз§’пјүгҖҚпјү

    гҖҗеҸӮиҖғгҖ‘

    пј‘з§’еҪ“гҒҹгӮҠгҒ®жңҖеӨ§иҰҒжұӮеҮҰзҗҶж•°(connections/sec)
пјқпјҲ гҖҢ MaxUserPort гҖҚ вҲ’ 1024 пјүпјҸ гҖҢ TcpTimedWaitDelay гҖҚ

    д»ҘдёҠгҒ§гҒҷгҖӮ
