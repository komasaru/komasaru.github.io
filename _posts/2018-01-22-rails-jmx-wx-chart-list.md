当方、気象庁防災情報 XML の情報を受信＆管理し、Web ページ上でも公開しております。

先日、一般にも公開が開始された各種天気図情報。その一覧ページも他の一覧ページ同様、当方 Web ページ上で公開を開始しました。

今回は、そのページの紹介のみ。

<!--more-->

### 1. 公開ページ

* [mk-mode SITE : 一覧 - 天気図情報（気象庁防災情報XML）](https://www.mk-mode.com/rails/jmaxml_list/lst_chart "mk-mode SITE : 一覧 - 天気図情報（気象庁防災情報XML）")

### 2. 注意事項

* 当コンテンツは、「気象庁」提供の「気象庁防災情報XML」受信記録・内容確認用。
* 対象の天気図情報は以下のとおり。
  + 地上実況図
  + 地上２４時間予想図
  + 地上４８時間予想図
  + アジア太平洋地上実況図
  + アジア太平洋海上悪天２４時間予想図
  + アジア太平洋海上悪天４８時間予想図
* 気象庁防災情報 XML での各種天気図情報の配信開始は 2018年1月17日。
* 「内容」ボタン押下で各情報の詳細な内容を表示。
* 等圧線等、緯度／経度の羅列を地図上に描画するとよいだろうが、現時点では未対応。
