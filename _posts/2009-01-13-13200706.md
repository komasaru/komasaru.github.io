僕の家での今現在のPC関連の環境について公開します。

主にパソコン３台とNAS１台という以下のような環境です。

主な用途はそれぞれの「主な用途」欄に記載しました。

何か知りたいことがあれば、きままに聞いてください。

![2009011319480000]({{site.baseurl}}/images/2009/01/2009011319480000.jpg "2009011319480000")
■Intel Atom マシン (完全自作、１枚目の写真の上の黒い小さいヤツ)
・ケース : Noah 800AL-BK
・マザーボード : Intel D945GCLF
・CPU : Intel Atom 230 (1.6GHz) (オンボード)
・MEMORY : DDR2-667 (PC2-5300) 1GB
・ハードディスク : MAXTOR 6V250F0 (250G SATAII300 7200)
・光学ドライブ: ソニーNECオプティアーク AD-7590A スリムDVDスーパーマルチドライブ
・OS : Fedora10 (Kernel: 2.6.27.9-159.fc10.i686)
・主な用途 : サーバマシン。主にWebサーバ・メールサーバ・FTPサーバ・ブログサーバ。

■Shuttle SD11G5 (PentiumM対応キューブ型ベアボーンキット、１枚目の写真の下の白いヤツ)
・CPU : Intel Pentium M 740 (1.73GHz)
・MEMORY : DDR2 SDRAM (PC2-4200) 1GB(512MB X 2)
・ハードディスク : WDC WD2500JD-22HBC0 (250G SATA150 7200)
・光学ドライブ: I-O DATA DVR-AN18GLB
・OS : Microsoft Windows XP Professional SP3
・主な用途 : メイン(普段使い)で使用。

![2009011319480001]({{site.baseurl}}/images/2009/01/2009011319480001.jpg "2009011319480001")
■Frontier G-BREAK (オーダーメイド。後に各種増設後、２枚目の写真のクリーム色の高さの高いヤツ)
・CPU : Intel Pentium 4 (1.6GHz)
・MEMORY : 512MB
・ハードディスク : MAXTOR 6L040L2 ( 40GB U133 7200)
MAXTOR 6Y160P0 (160GB U133 7200)
・光学ドライブ: CD-ROM 52X/AKH
・FDドライブ : MITSUMI
・OS : Microsoft Windows XP Professional SP3
・主な用途 : 昔メインで使用。今は、TVキャプチャボードを増設して録画専用のマシン。

![2009011320030000]({{site.baseurl}}/images/2009/01/2009011320030000.jpg "2009011320030000")
■NAS (３枚目の写真)
・ケース : 玄人志向 KURO-BOX/HG
・ハードディスク : WDC WD3200JB-22KFA0
・OS : Fedora8 (プリインストールのLinuxから乗せ替え)
・主な用途 : 録画PCからのmpegデータ保存用。メインPCからそれらを閲覧。各種バックアップファイル保存用。

■ディスプレイ
・MITSHUBISHI Diamondcrysta RDT1710S
・１台のディスプレイをディスプレイのDVI入力とアナログ入力で切り替えたり、アナログ入力を録画PCとサーバマシンで切り替えたりして使用しています。

■その他、外付けだの、USB接続のWebカメラなどもありますけど。

※今、たくらんでいる事が１つ。

それは、いまだに自宅ではIntel Dual Core 等の新しいCPUのマシンが無いので、新たに１台自作しようかと。。。

スペックはCPUはDualCore か QuadCore、マザーはASUS の P５Q−Deluxe で。

もちろん今使っている録画マシンはやめて、地デジ鑑賞＆録画もできるようなものを・・・

少しずつ、パーツを入手しようかともくろんでいるところです。

皆さんは、どんなPC環境ですか？
