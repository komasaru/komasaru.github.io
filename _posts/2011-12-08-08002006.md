<!--more-->

この [OpenCV](http://sourceforge.net/projects/opencvlibrary/ "OpenCV") とは、インテル社が開発したオープンソースのライブラリ集（C/C++）で、画像から人間の顔を検出して自動でモザイクをかけたり、動く物体を自動で追跡したりと様々な処理が可能です。

実際に Ruby で [OpenCV](http://sourceforge.net/projects/opencvlibrary/ "OpenCV") を使用するには、本家の [OpenCV](http://sourceforge.net/projects/opencvlibrary/ "OpenCV") 導入とは別に Ruby/OpenCV が必要のようですが、今回はとりあえず本家の [OpenCV](http://sourceforge.net/projects/opencvlibrary/ "OpenCV") を導入しました。

当方の Linux サーバは Scientific Linux なので、yum を使えば [OpenCV](http://sourceforge.net/projects/opencvlibrary/ "OpenCV") パッケージはインストールできますが、リポジトリの関係でバージョンが多少古いです。
※当記事執筆時点で [OpenCV](http://sourceforge.net/projects/opencvlibrary/ "OpenCV") の最新は 2.3.1a なのに対して、yum でのインストールバージョンは 2.0。

そこで、ソースからのインストールを試みました。

### OpenCV 導入手順

[OpenCV](http://sourceforge.net/projects/opencvlibrary/ "OpenCV") の最新は 2.3.1a ですが、当方の環境では 2.3.1a では cmake でエラーとなるので、今回は 2.3.0 をインストールしました。

#### １．cmake のインストール

コンパイルに必要な cmake をインストールする。

``` bash 
# yum -y install cmake
```

#### ２．各種ライブラリのインストール

画像データを扱うためのライブラリ（JPEG・PNG・TIFF・圧縮）をインストールする。

``` bash 
# yum -y install libjpeg libjpeg-devel
# yum -y install libpng libpng-devel
# yum -y install libtiff libtiff-devel
# yum -y install zlib zlib-devel
```

#### ３．OpenCV のダウンロード・展開

[OpenCV](http://sourceforge.net/projects/opencvlibrary/ "OpenCV") アーカイブをダウンロード後、アーカイブを展開し、リリース用バイナリの配置場所（ディレクトリ）を作成します。

``` bash 
# wget http://sourceforge.net/projects/opencvlibrary/files/opencv-unix/2.3/OpenCV-2.3.0.tar.bz2/download
# tar jxvf OpenCV-2.3.0.tar.bz2
# cd OpenCV-2.3.0
# mkdir release
# cd release
```

#### ４．OpenCV のインストール

configure（Makeファイル作成）、make（コンパイル）、make install（インストール）を行います。

``` bash 
# cmake -D BUILD_EXAMPLES=ON -D CMAKE_BUILD_TYPE=RELEASE -D CMAKE_INSTALL_PREFIX=/usr/local -D BUILD_PYTHON_SUPPORT=ON -D WITH_1394=OFF -D ENABLE_OPENMP=ON ../
# make
# make install
```
