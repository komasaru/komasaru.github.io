今日は、
Ｆｅｄｏｒａ１１をインストールし、
ファイルサーバ導入済みの玄箱ＨＧを
Ｆｅｄｏｒａ１２にアップグレードしてみました。

※せっかく、Ｆｅｄｏｒａ１２までリリースされているんで。

以下、今回やったことです。

### 作業記録

#### １．玄箱ＨＧへの接続

ローカルマシンから KURO-BOX/HG へリモート接続し、rootユーザになる。
（当方は、「 Poderosa 」というのを使用しています）

#### ２．Ｆｅｄｏｒａパッケージインストール

「 http://download.fedora.redhat.com/pub/fedora/linux/releases/12/Fedora/ppc/os/Packages/ 」にある
「 fedora-release-12-1.noarch.rpm 」と
「 fedora-release-notes-12.0.0-4.fc12.noarch.rpm 」
をインストールする。

``` text
[root@KURO-BOX ~]# rpm -Uvh http://download.fedora.redhat.com/pub/fedora/linux/releases/12/Fedora/ppc/os/Packages/fedora-release-12-1.noarch.rpm http://download.fedora.redhat.com/pub/fedora/linux/releases/12/Fedora/ppc/os/Packages/fedora-release-notes-12.0.0-4.fc12.noarch.rpm

・・・　途中省略　・・・

警告: /var/tmp/rpm-tmp.FuQBrs: ヘッダ V3 RSA/SHA256 signature: NOKEY, key ID 57bbccba
エラー: 依存性の欠如:
        xdg-utils は fedora-release-notes-12.0.0-4.fc12.noarch に必要とされています
[root@KURO-BOX ~]#
```依存性エラー！

#### ３．「 xdg-utils 」インストール

``` text
[root@KURO-BOX ~]# yum -y install xdg-utils
```

#### ４．再度、Ｆｅｄｏｒａパッケージインストール

``` text
[root@KURO-BOX ~]# rpm -Uvh http://download.fedora.redhat.com/pub/fedora/linux/releases/12/Fedora/ppc/os/Packages/fedora-release-12-1.noarch.rpm http://download.fedora.redhat.com/pub/fedora/linux/releases/12/Fedora/ppc/os/Packages/fedora-release-notes-12.0.0-4.fc12.noarch.rpm
http://download.fedora.redhat.com/pub/fedora/linux/releases/12/Fedora/ppc/os/Packages/fedora-release-12-1.noarch.rpm を取得中
http://download.fedora.redhat.com/pub/fedora/linux/releases/12/Fedora/ppc/os/Packages/fedora-release-notes-12.0.0-4.fc12.noarch.rpm を取得中
警告: /var/tmp/rpm-tmp.FhLx56: ヘッダ V3 RSA/SHA256 signature: NOKEY, key ID 57bbccba
準備中...                ########################################### [100%]
   1:fedora-release-notes   ########################################### [ 50%]
   2:fedora-release         ########################################### [100%]
[root@KURO-BOX ~]#
```インストール成功！

#### ５．キャッシュの整理

``` text
[root@KURO-BOX ~]# yum clean all
すべて掃除しています
[root@KURO-BOX ~]#
```

#### ６．アップグレード

``` text
[root@KURO-BOX ~]# yum -y upgrade
```当方の環境では、５０分くらいで無事完了！

#### ７．リブート

念のため、玄箱ＨＧをリブートする。

#### ８．バージョン確認

再度リモート接続しrootユーザになり、以下のように入力してみる。

``` text
[root@KURO-BOX ~]# cat /etc/fedora-release
Fedora release 12 (Constantine)　←　Ｆｅｄｏｒａ１２になってる
[root@KURO-BOX ~]#
```

---

作業は以上です。

当分、うちの「 玄箱ＨＧ 」は
これで運用します。。。

ではまた。
