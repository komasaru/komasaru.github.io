    Fortran - ヒープ生成（上方・下方移動）！

    Updated: 2019-03-05 00:20


    Fortran 95 でヒープ（上方移動・下方移動）のアルゴリズムを実装してみました。

    0. 前提条件

    • LMDE 3 (Linux Mint Debian Edition 3; 64bit) での作業を想定。
    • GCC 6.3.0 (GFortran 6.3.0) でのコンパイルを想定。

    1. アルゴリズムについて

    当ブログ過去記事を参照のこと。

    • C++ - ヒープ生成（上方移動）！
    • C++ - ヒープ生成（下方移動）！
    • Ruby - ヒープ生成（上方・下方移動）！
    • Python - ヒープ生成（上方・下方移動）！

    2. ソースコードの作成

    以下は上方移動。

    File: heap_upward.f95

    • Gist - Fortran 95 source code to generate a heap by upward method.

    以下は下方移動。

    File: heap_downward.f95

    • Gist - Fortran 95 source code to generate a heap by downward method.

    3. ソースコードのコンパイル

    $ gfortran -o heap_upward heap_upward.f95
$ gfortran -o heap_downward heap_downward.f95
