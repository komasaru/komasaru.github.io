    Fortran - 正規乱数（ボックス＝ミューラー法）！

    Updated: 2019-02-17 00:20


    Fortran 95 で、 Box-Muller 法を使って正規乱数を生成してみました。

    0. 前提条件

    • LMDE 3 (Linux Mint Debian Edition 3; 64bit) での作業を想定。
    • GCC 6.3.0 (GFortran 6.3.0) でのコンパイルを想定。

    1. アルゴリズムについて

    当ブログ過去記事を参照のこと。

    • C++ - 正規乱数（ボックス＝ミューラー法）！
    • Ruby - 正規乱数（ボックス＝ミューラー法）！
    • Python - 正規乱数（ボックス＝ミューラー法）！

    2. ソースコードの作成

    File: rndnum_bm.f95

    • Gist - Fortran 95 source code to generate random numbers by box-muller’s method.

    3. ソースコードのコンパイル

    $ gfortran -o rndnum_bm rndnum_bm.f95

    4. 動作確認

    $ ./rndnum_bm
